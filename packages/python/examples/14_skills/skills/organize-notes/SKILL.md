## Instructions

1. Read all note files from the working directory
2. Identify common themes and topics across the notes
3. Group related content into logical sections with headers
4. Write the organized output to a new file called `organized-notes.md`
5. Provide a summary of what was organized

## Output Format

Use Markdown with clear headers:

```markdown
# Organized Notes

## Topic 1
- Point from note A
- Related point from note B

## Topic 2
...
```
